# 一灯学堂Vue Pc端脚手架

[![NPM version](https://badge.fury.io/js/yog.png)](http://badge.fury.io/js/yog) 

An koa2-based Node.js web application bootstrapping module,with Vue2.

### install

on Linux/Unix

```bash
$ npm install -g yarn
```

then open `cmd`

```bash
yarn install
```
